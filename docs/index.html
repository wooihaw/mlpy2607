What you'll cover

A three-day journey through the ML pipeline

The course is organised so each stage builds on the last — from preparing messy data, to training models, to tuning them for real performance.

Day 1

Data & Foundations

  • Python & JupyterLab refresher
  • Loading & exploring datasets (EDA)
  • Cleaning, missing values & outliers
  • Encoding, scaling & feature engineering
Day 2

Models & Evaluation

  • Feature selection techniques
  • Train/test split & cross-validation
  • Regression & classification algorithms
  • Ensembles & neural networks
Day 3

Tuning & Unsupervised

  • Hyperparameter tuning & pipelines
  • K-Means & Gaussian Mixture clustering
  • PCA & dimensionality reduction
  • Capstone hands-on projects

Get started in minutes

Clone, sync, and launch

The project uses uv for fast, reproducible environments. No git? Download the ZIP from GitHub and skip step 1.

# 1 · get the code
git clone https://github.com/wooihaw/mlpy2607.git
cd mlpy2607

# 2 · restore the exact dependencies
uv sync

# 3 · launch JupyterLab and start learning
uv run jupyter-lab
